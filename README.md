# Linked List Node Deletion

## Project Overview
This Java project demonstrates working with a singly linked list.  
The program deletes `n` nodes after keeping `m` nodes repeatedly throughout the list.

## Features
- Create a linked list of random integers
- Print the original linked list
- Delete `n` nodes after every `m` nodes
- Print the modified linked list

## Classes
- `Node` – Represents a node in the linked list
- `Assignment1test` – Main class that contains:
  - `deleteNodes()` method
  - `main()` method to run the program

## Usage
1. Run `Assignment1test.java`
2. The program generates a random linked list
3. Random values for `m` and `n` are used (or can be manually set)
4. The modified linked list is printed after deletion
